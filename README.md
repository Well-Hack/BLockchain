📌 Sobre o Projeto
Este projeto demonstra como criar, configurar e interagir com uma carteira Bitcoin utilizando o Electrum integrado com Node.js. O objetivo é fornecer uma implementação educacional de gerenciamento de carteira, incluindo geração de endereços, consulta de saldo e assinatura de transações.

🛠️ Tecnologias Utilizadas

Electrum — Carteira Bitcoin leve e confiável
Node.js 18+ — Linguagem principal
npm — Gerenciador de pacotes
Bitcoin Testnet — Rede de testes (sem valor real)

Antes de começar, certifique-se de ter instalado:

Node.js 18+
npm
Electrum

🔐 Segurança

⚠️ AVISO IMPORTANTE


Nunca compartilhe sua seed phrase (frase-semente) com ninguém.
Nunca armazene sua chave privada em texto puro no código ou repositório.
Use sempre a Bitcoin Testnet para testes antes de usar fundos reais.
Faça backup da sua carteira regularmente.
Este projeto é para fins educacionais. Não use em produção sem auditoria de segurança.
